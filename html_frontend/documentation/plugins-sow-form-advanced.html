Advanced Lists/expanders

  • Preview

1. required attribute is removed when containers are hidden so you can submit the form having required hidden fields!
2. disable attribute is added when containers are hidden. .js-form-advanced-list-ignore class added on element will cancel this feature!
3. AJAX content is loaded once, on option click! SOW : Ajax : Content can also used to load the content (on page load).
