Vendor : Datatable

DataTables is a plug-in for the jQuery Javascript library. It is a highly flexible tool, built upon the foundations of progressive enhancement, that adds all of these advanced features to any HTML table.

  • License: MIT
  • https://datatables.net/
  • src/js/vendors/vendor.datatables.js

Datatable Info

  • Preview
  • NPM info

NOTE: Smarty Datatable using HTML attributes is curently in Beta!
You can create your own datatable by passing this attribute: data-custom="true"
From this point, you handle the entire script - plugin is loaded as long you keep .table-datatable class!

Using additional attributes to quick add/overwrite default config:
data-custom-config='{"paging":false, "searching":false}'

Using javascript to extend/overwrite current Smarty config:
(this example will hide pagination and searching)

                        <script>
                        var dataTableExtend = {
                          paging: false,
                          searching: false,
                        };
                      </script>
