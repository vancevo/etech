            RTL ARROWS
            
Will turn the oposite direction on RTL, without a need to edit the html code


               <i class="fi fi-arrow-start"></i> 

               <i class="fi fi-arrow-end"></i> 

            
               <i class="fi fi-arrow-start-slim"></i> 

               <i class="fi fi-arrow-end-slim"></i> 


               <i class="fi fi-arrow-start-full"></i> 

               <i class="fi fi-arrow-end-full"></i> 


          


          

            Example: <i class="fi fi-orbit fi-spin"></i>
               
               
               
               
          


          


          

            Please see Utilities for available classes (font size & colors).

            Example: <i class="fi fi-star-empty small text-danger"></i>
