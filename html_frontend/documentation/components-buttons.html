Icon correction

When using icons, the text is too close because there is no margin.
To avoid using &nbsp;, Smarty will add required margins to icons and few corrections are needed in some situations:

1. Icon only, no text

class .m-0 added directly to the icon

  bad      good      nothing required 
                  
                    <!-- bad -->
                    <a href="#" class="btn btn-primary">
                      <i class="fi fi-check"></i>
                    </a>

                    <!-- good -->
                    <a href="#" class="btn btn-primary">
                      <i class="fi fi-check m-0"></i>
                    </a>

                    <!-- nothing required -->
                    <a href="#" class="btn btn-primary rounded-circle">
                      <i class="fi fi-check"></i>
                    </a>
2. Icon after text only

text is wrapped inside <span>

Bad    Good    Nothing required 
                  
                    <!-- bad -->
                    <a href="#" class="btn btn-primary">
                      Bad
                      <i class="fi fi-arrow-end"></i>
                    </a>

                    <!-- good -->
                    <a href="#" class="btn btn-primary">
                      <span>Good</span>
                      <i class="fi fi-arrow-end"></i>
                    </a>

                    <!-- nothing required -->
                    <a href="#" class="btn btn-primary">
                      <i class="fi fi-arrow-end"></i>
                      Nothing required
                    </a>
