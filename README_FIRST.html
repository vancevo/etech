Before starting

– local webserver required –

Smarty load plugins via ajax, so a local webserver is required by the browser!
GULP is optional but if you use it, you can directly start a local webserver: 

gulp webserver --root html_frontend

- or - for the admin:

gulp webserver --root html_admin

Please read more in the official documentation!
